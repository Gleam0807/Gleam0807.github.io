> 풀이 1 ( 중첩 반복문 )

```yaml

process.stdin.setEncoding('utf8');
process.stdin.on('data', data => {
    const n = data.split(" ");
    const a = Number(n[0]), b = Number(n[1]);
    for(let i=0; i<b; i++) {
        let c = "";
        for(let j=0; j<a; j++) {
            c += "*";
        }
        console.log(c);
    }
});

```

<br/>


> 풀이 2 ( repeat 함수 이용 )

```yaml

process.stdin.setEncoding('utf8');
process.stdin.on('data', data => {
    const n = data.split(" ");
    const a = Number(n[0]), b = Number(n[1]);
    const star = "*".repeat(a);
    for(let i=0; i < b; i++){
        console.log(star);
    }
});

```
